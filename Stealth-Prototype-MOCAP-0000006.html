Stealth Prototype MOCAP

Concept - Design - Programming - Modelling - Level Design

Project #6 Roman Wecker

The Backstory

This prototype was created during my studies in the subject 'Controlling Virtual Characters'. In a team of five people, we created 3D models, rigged them, recorded motion capture footage in the motion capture lab at Kempten University of Applied Sciences and imported it into Unreal Engine 4. It is a stealth game with quicktime events.

We used Blender to create and rig the 3D models. The MOCAP data was recorded with the OptiTrack system from Motive. The game was entirely build using Blueprints in Unreal Engine 4.
